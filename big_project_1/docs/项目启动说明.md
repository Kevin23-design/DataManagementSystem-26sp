# 基于 MongoDB 的问卷系统 - 启动说明文档

这是一份简易指南，指导你如何在本地环境中启动并运行整个问卷系统项目。

## 1. 环境准备

在开始之前，请确保你的电脑上已经安装了以下软件：
- **Node.js** (推荐 v18 或以上版本)
- **MongoDB** (推荐 v6.0 或以上版本)

## 2. 启动 MongoDB 服务

本系统依赖 MongoDB 数据库存储数据。在启动后端服务器前，必须先启动 MongoDB。

如果你已经将 MongoDB 注册为系统服务且设为开机自启，可以跳过此步；否则，请打开一个终端并运行 MongoDB：
```bash
mongod --dbpath <你的数据存储目录路径>
```
*例如： `mongod --dbpath F:\CodeSoftware\MongoDB\data`*

确保看到终端输出中包含 `Waiting for connections on port 27017`，这表示数据库已成功启动。

## 3. 安装后端依赖

接下来，进入项目后端服务的目录并安装所需的 npm 包。

1. 打开一个**新的终端或命令行窗口**（不要覆盖刚才运行 MongoDB 的窗口）。
2. 导航到项目的 `server` 文件夹：
   ```bash
   cd f:\code\DataManagementSystem\big_project_1\server
   ```
3. 执行 npm 安装命令：
   ```bash
   npm install
   ```

## 4. 环境变量配置 (可选)

项目在 `server` 文件夹下提供了一个预设的 `.env` 配置文件，包含了默认设定：
```text
PORT=3000
MONGODB_URI=mongodb://localhost:27017/survey_system
JWT_SECRET=survey_system_secret_key_2026
```
默认配置已能开箱即用，如你的端口号冲突或数据库设置了密码，请自行在此修改。

## 5. 启动项目后端

在依赖安装完毕的终端（`server` 文件夹）内，运行启动命令：
```bash
npm start
```
如果启动成功，你将会在控制台连续看到两行提示：
```text
MongoDB 已连接: localhost
服务器运行在 http://localhost:3000
```

## 6. 访问问卷系统网页端

本项目的前端为存放在 `public` 文件夹下的纯静态页面，已交由 Express 后端统一托管。

请打开你的浏览器，并在地址栏输入：
👉 **http://localhost:3000**

- **初次进入**：会展示登录和注册页面，你可以点击“立即注册”创建一个新账号（如：`myuser1`，密码 `test123456`）。
- **注册成功后**：系统会自动跳转到《我的问卷》看板，并允许你点击右上方 "+ 创建问卷" 按钮开始设计你的问卷。
